================================================================================
                    IF YOU GET STUCK
================================================================================

1. Stripe env vars not working?
   → Check exact variable names match: STRIPE_SECRET_KEY (no typos)
   → Make sure they're in Vercel Environment Variables, not .env
   → Vercel needs to rebuild after adding env vars

2. Payment redirects to error?
   → Check STRIPE_PRICE_* IDs are correct
   → Check products are set to "subscription" mode (not one-time)
   → Check Stripe account is in "live" mode if using live keys

3. Webhook doesn't fire?
   → Check endpoint URL is exactly right
   → Check webhook secret matches STRIPE_WEBHOOK_SECRET
   → Check Stripe can reach your domain (firewall, etc)
   → Check Stripe webhook logs for delivery attempts

4. User tier doesn't update?
   → Check webhook was received (check Stripe logs)
   → Check server logs in Vercel for errors
   → Check database connection string is correct
   → Check user_id in webhook matches database user
