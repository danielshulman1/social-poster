## 🚀 Quick Start (2 Hours)

### Step 1: Create OAuth Apps (30 minutes)

You'll:
1. Go to X/Twitter, TikTok, YouTube, Pinterest developer consoles

3. Add redirect URLs
4. Copy credentials

### Step 2: Add Credentials (5 minutes)

Update `.env.livecheck`:
```bash
TWITTER_CLIENT_ID="..."

TIKTOK_CLIENT_ID="..."
TIKTOK_CLIENT_SECRET="..."
YOUTUBE_CLIENT_ID="..."
YOUTUBE_CLIENT_SECRET="..."
PINTEREST_CLIENT_ID="..."
PINTEREST_CLIENT_SECRET="..."
```

### Step 3: Deploy to Vercel (5 minutes)

Follow: [`DEPLOYMENT_STEPS.md`](./DEPLOYMENT_STEPS.md)

Add same credentials to Vercel environment variables.

### Step 4: Test OAuth (30 minutes)

Follow: [`OAUTH_TEST_GUIDE.md`](./OAUTH_TEST_GUIDE.md)

Verify all 4 platforms work.

### Step 5: Add UI Buttons (15 minutes)

Add "Connect" buttons to your Connections page.
